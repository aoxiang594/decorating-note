## 需求
基本需求
* 安全耐用。换电线，水管。
* 细节的隐藏。电箱，调制解调器，路由器，网线。

  * 放调料的架子
  * 放碗盆，米饭，面粉，五谷杂粮的橱柜

  * 净水器

## 插座的预留
* 厨房。 台面至少3个插座。 水槽下面至少2个。
* 卫生间至少3个。

## 设计师
> 设计出满足你的预算和需求的设计方案。
