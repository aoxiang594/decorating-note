## 需求
基本需求
* 安全耐用。换电线，水管。
* 健康。 少甲醛，纯净的水，空气。
* 细节的隐藏。电箱，调制解调器，路由器，网线。

## 插座的预留
* 厨房。 台面至少3个插座。 水槽下面至少2个。
* 卫生间至少3个。

## 好物
* 虹吸式地漏。排水量是普通的三倍。
* 榻榻米。很多做榻榻米床，内部方便收纳，贴墙做，节省空间，有的榻榻米中间还带升降功能（升降机）。
